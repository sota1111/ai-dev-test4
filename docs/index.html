TimeQuest Ops

AIエージェントが、あなたの歩く街を冒険の舞台に変える。歴史、文化、発見が詰まった次世代の街歩きプラットフォーム。

コンセプトを見る

The Problem

🏛️
資源の埋没

地域の貴重な歴史や文化資源が、日常の中に埋もれて見過ごされています。

📍
観光の偏り

有名な観光スポットに人が集中し、路地裏や地域の魅力が十分に伝わっていません。

♿
視点の欠如

防災やバリアフリー、歴史的な背景など、多角的な視点で街を見る機会が不足しています。

The Solution

Gemini がリアルタイムに街の文脈を読み解き、パーソナライズされたクエストを生成します。

AI Agents

🗺️

Quest Planner

現在地と周辺DBから、その場に最適なクエストを設計。

🎙️

Quest Guide

ユーザーの投稿をVisionで解析し、ヒントや解説を提供。

🛡️

Quest QA

生成された内容の安全性、事実性、表現を厳格にチェック。

📈

Quest Ops

利用データから「盛り上がったポイント」を分析し改善を提案。

MVP Features

📡

現在地クエスト

GPS連動で、今いる場所が即座に冒険のフィールドに。

📷

写真判定
